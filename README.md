# Torder

Um protótipo criado para facilitar a reserva de mesas de um restaurante ou lanchonete, e diretamente do 

próprio app, também conseguir visualizar o cardápio do lugar.

Linguagem

Javascript

Tecnologias

React Native, Mongodb
